Sudoku - Get Started!

  • Choose the 'Puzzle' page to attempt the sudoku puzzle.
  • Choose the 'Solved' page to check out the solution.
  • Go ahead and submit the solved puzzle and the algorithm will verify the 'certificate'.

Rules

  • Start with a 9x9 grid of squares made up of 3x3 grids of squares.
  • Some squares/cells will already have numbers entered in them.
  • Fill in empty squares such that all rows, columns, and 3x3 grids contain the numbers 1-9 EXACTLY ONCE.

Verification Algorithm

  • For time complexity analysis, let n = 3.
  • The first step in verification is looping through the n^4 cells to organize them in a matrix of row elements - this step is completed in O(c_1n^4) time
  • Next, we loop through the n^4 cells to organize them in a matrix of column elements - this step is completed in O(c_2n^4) time.
  • Finally, we loop through the n^4 cells to organize them in a matrix smaller of 3 x 3 grid elements - this step is completed in O(c_3n^4) time.
  • Note, the time complexity for the above operations is n^4, because we only process each of the n^4 exactly once (with some constant time operations for each element).
  • Next, we attempt to verify the three matrices as viable solutions for the Sudoku puzzle.
  • For each of the matrices of n^4 elements, we loop through each of the n^4 elements a maximum of twice and perform constant time operations for each pass through the elements.
  • Finally, we interpret the results of the verification step and populate some messaging to return to the user in O(1) time.
  • The overall time complexity is as follows: O(n^4) = O(c_1n^4) + O(c_2n^4) + O(c_3 n^4) + O(c_4n^4 + c_5n^4) + O(1)
